Политика конфиденциальности

Приложение «РендерАИ» (Мебель AI) — Последнее обновление: апрель 2026

1. Какие данные мы собираем

Приложение РендерАИ работает локально на вашем устройстве. Мы не собираем персональные данные, не требуем регистрации и не используем систему аккаунтов.

Изображения, которые вы загружаете для обработки, отправляются на серверы обработки изображений исключительно для создания визуализаций и не сохраняются на серверах после завершения обработки.

2. Хранение данных

Все проекты, шаблоны и результаты хранятся локально на вашем устройстве. При удалении приложения все данные удаляются вместе с ним.

3. Камера и микрофон

Приложение запрашивает доступ к камере для съёмки фото мебели, к галерее для загрузки изображений, и к микрофону для голосового ввода описания дизайна. Эти разрешения используются только по вашему запросу.

4. Передача данных третьим лицам

Мы не продаём, не передаём и не раскрываем ваши данные третьим лицам. Изображения обрабатываются через защищённое соединение и не хранятся на серверах.

5. Аналитика

Приложение не использует трекеры, аналитические системы или рекламные SDK.

6. Дети

Приложение не предназначено для использования лицами младше 13 лет.

7. Изменения в политике

Мы можем обновлять данную политику. Актуальная версия всегда доступна по этому адресу.
